# Medidor de Distância com ESP32 e Sensor Ultrassônico

Este projeto utiliza um módulo **ESP32** e um sensor ultrassônico **HC-SR04** para medir distâncias em tempo real, e emitr um aviso de proximidade por meio de um LED. Os dados são exibidos no Monitor Serial, podendo ser facilmente adaptados para displays OLED ou dashboards na nuvem.

## Funcionalidades

* Medição de distância em centímetros (cm).
* Aviso por meio de LED
* Leitura precisa utilizando pulsos de som.
* Fácil integração com outros sistemas de IoT via Wi-Fi/Bluetooth (nativos da ESP32).

## Componentes Necessários

| Componente | Quantidade |
| :--- | :--- |
| ESP32 DevKit V1 | 1 |
| Sensor Ultrassônico HC-SR04 | 1 |
| Cabos Jumper (Macho-Fêmea) | 5 |
| Protoboard | 1 |
| LED | 1 |
| Resistor | 2 |
